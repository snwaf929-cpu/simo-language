# Roadmap

Simo 0.5 provides a usable console language and web/app compiler. The remaining work toward a stable 1.0 ecosystem is:

1. Language server protocol implementation for live diagnostics and completion.
2. Published VS Code extension package.
3. Source maps and browser stack traces mapped back to `.simo` lines.
4. Component syntax, page routing, and reusable UI packages.
5. Secure native APIs for desktop files, dialogs, notifications, and system integration.
6. Automated platform installers and signed release artifacts.
7. Package registry and dependency lock file.
8. Native mobile packaging.
9. Separate Canvas/WebGL game target, physics, audio, and input systems.
10. Performance optimization, compatibility policy, and a formal 1.0 specification.

Backward compatibility is not guaranteed before 1.0.
