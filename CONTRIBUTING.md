# Contributing

1. Use Python 3.11 or newer.
2. Install the project in editable mode: `python -m pip install -e .`.
3. Add tests for syntax, runtime, or compiler behavior.
4. Run `python -m unittest discover -s tests -v`.
5. Run the console and web examples before submitting changes.

Language changes should update `docs/LANGUAGE.md`, compiler behavior, interpreter behavior where applicable, and tests.
